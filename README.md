## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd cybersecurity-compass
   ```

2. **Start local server**
   ```bash
   python -m http.server 8000
   # or
   npx serve .
   ```

3. **Open in browser**
   ```
   http://localhost:8000
   ```

4. **Explore the features**
   - Navigate through different sections
   - Try the assessment tool
   - Use keyboard shortcuts
   - Open console for debug commands

## 🔍 Troubleshooting

### **Common Issues**
1. **Charts not loading**: Check Chart.js CDN connection
2. **Icons missing**: Verify Lucide icons CDN
3. **Animations slow**: Reduce particle count
4. **Console errors**: Run `debugAll()` for diagnostics

### **Debug Commands**
```javascript
// Check system status
debugAll()

// Test specific features
testAssessment()

// Clear local storage
localStorage.clear()

// Reset to defaults
```
